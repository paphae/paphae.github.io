java-数据结构

2020.03.20 Paphae 工具

类型介绍

数据元素相互之间的关系称为结构

具备关联关系的非松散集合性结构主要有三种：

  • 线性表
    • 结点按逻辑关系依次排列形成一个“锁链”
  • 树
    • 具有分支、层次特性,其形态有点象自然界中的树
  • 图
    • 结点按逻辑关系互相缠绕,任何两个结点都可以邻接
