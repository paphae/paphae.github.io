系统相关的路径分隔符，为方便起见，以字符串形式表示。
static char pathSeparatorChar

FileItem Api

修饰符和类型 方法和说明
void delete()删除文件项的基础存储，包括删除任何关联的临时磁盘文件。
byte[] get()以字节数组形式返回文件项的内容。
String getContentType()返回浏览器传递或null未定义的内容类型。
String getFieldName()以多部分形式返回与此文件项相对应的字段名称。
InputStream getInputStream()返回一个InputStream可用于检索文件内容的。
String getName()返回浏览器（或其他客户端软件）提供的客户端文件系统中的原始文件名。
OutputStream getOutputStream()返回OutputStream可用于存储文件内容的。
long getSize()返回文件项的大小。
String getString()使用默认字符编码将文件项的内容作为String返回。
String getString(String encoding)使用指定的编码，以字符串形式返回文件项的内容。
boolean isFormField()确定FileItem实例是否代表简单表单字段。
boolean isInMemory()提供有关是否从内存中读取文件内容的提示。
void setFieldName(String name)设置用于引用此文件项的字段名称。
void setFormField(boolean state)指定FileItem实例是否代表简单表单字段。
void write(File file)一种将上传的项目写入磁盘的简便方法。
