SpringBoot-初识

2020.07.10 Paphae 框架

Spring Boot的主要优点：

  • 为所有Spring开发者更快的入门
  • 开箱即用，提供各种默认配置来简化项目配置
  • 内嵌式容器简化Web项目
  • 没有冗余代码生成和XML配置的要求

编写相关的Controller

注意：

  • springmvc 或spring 的使用我们使用注解后。要在spring或springmvc的配置文件中。
  • 配置要扫描的包。springboot 我们没有设置扫描包。
  • springboot设置自动扫描 程序主应用HelloWorldMainApplicaiton所在位置，它下面或同级的包。
  • @ResponseBody放在控制器类上。该控制所有的方法的返回类型都是json。
  • @RestController ==@ResponseBody+@Controller

简化部署

  • 在pom文件进行导入build。

  • maven –> 项目名称 –> lifecycle –> 双击packaage

  • 在target文件夹下生成该项目的jar包。

  • 避免端口号冲突。要确保idea中的程序停止运行。

  • 在jar包所在目录下cmd。

  • java -jar springboot-demo01-1.0-SNAPSHOT.jar 执行。

  • 将这个应用打成jar包，直接使用java -jar的命令进行执行；

  • 如果是springboot进行应用部署的话只需要jdk和相应数据库。
