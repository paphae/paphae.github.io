Maven项目结构

一个使用Maven管理的普通的Java项目，它的目录结构默认如下：

maven-project
├── pom.xml
├── src
│   ├── main
│   │   ├── java
│   │   └── resources
│   └── test
│   ├── java
│   └── resources
└── target

项目的根目录maven-project是项目名

它有一个项目描述文件pom.xml

存放Java源码的目录是src/main/java

存放资源文件的目录是src/main/resources

存放测试源码的目录是src/test/java

存放测试资源的目录是src/test/resources

最后，所有编译、打包生成的文件都放在target目录里。
